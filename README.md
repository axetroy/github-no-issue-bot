# github-no-issue-bot

> A GitHub App built with [Probot](https://github.com/probot/probot) that close the issue of non-owners

## Setup

```sh
# Install dependencies
yarn

# Create an new environment file
cp .env.example .env

# Update environment file
vi .env

# Run in development
npm run dev
```
